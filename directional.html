FizzBuzz: Up and Down

Controlăm direcția! Programul trebuie să urce până la 20 și să coboare înapoi la 1 fără oprire.

1. Bucle (while): Una pentru urcare, una pentru coborâre.
2. Incrementare: i = i + 1 (Urcăm la deal 🏔️)
3. Decrementare: i = i - 1 (Coborâm la vale ⛷️)
URCARE
⬆️
1
Vezi cele două bucle C++ ↓

Structura cu Două Bucle While

#include <iostream>
using namespace std;

int main() {
    int i = 1;

    // PRIMA BUCLĂ: Urcăm de la 1 la 20
    while (i <= 20) {
        if (i % 3 == 0) cout << "Fizz" << endl;
        else cout << i << endl;
        
        i = i + 1; // Mergem înainte
    }

    // Resetăm punctul de plecare pentru coborâre
    i = 19; 

    // A DOUA BUCLĂ: Coborâm de la 19 la 1
    while (i >= 1) {
        if (i % 3 == 0) cout << "Fizz" << endl;
        else cout << i << endl;
        
        i = i - 1; // Mergem înapoi
    }

    return 0;
}
