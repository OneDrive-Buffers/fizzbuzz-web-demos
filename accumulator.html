FizzBuzz Accumulator

Urmărește cum se schimbă variabila scor în funcție de numărul verificat!

• Dacă numărul se împarte la 3: Scor +3 🔵
• Dacă numărul se împarte la 5: Scor +5 🔵
• Dacă se împarte la amândouă (15): Scor -10 🔴
Verificăm numărul:
1
0
SCOR TOTAL
Vezi codul C++ ↓

Soluția în C++

#include <iostream>
using namespace std;

int main() {
    int scor = 0; // Punctul de start

    for (int i = 1; i <= 20; i++) {
        
        if (i % 3 == 0 && i % 5 == 0) {
            scor = scor - 10; // Penalizare FizzBuzz
        } 
        else if (i % 3 == 0) {
            scor = scor + 3;  // Bonus Fizz
        } 
        else if (i % 5 == 0) {
            scor = scor + 5;  // Bonus Buzz
        }

        // Afișăm scorul la fiecare pas
        cout << "Numar: " << i << " | Scor: " << scor << endl;
    }

    return 0;
}
