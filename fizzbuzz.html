FizzBuzz Clasic

Problema clasică de filtru pentru programatori. Spunem cuvinte magice în loc de numere!

• Se împarte la 3? Spunem Fizz 🔵
• Se împarte la 5? Spunem Buzz 🟡
• Se împarte la amândouă (15)? Spunem FizzBuzz 🟠

Codul Sursă C++

#include <iostream>
using namespace std;

int main() {
    // Parcurgem numerele de la 1 la 20
    for (int i = 1; i <= 20; i++) {

        if (i % 3 == 0 && i % 5 == 0) {
            cout << "FizzBuzz" << endl;
        } 
        else if (i % 3 == 0) {
            cout << "Fizz" << endl;
        } 
        else if (i % 5 == 0) {
            cout << "Buzz" << endl;
        } 
        else {
            cout << i << endl;
        }
    }

    return 0;
}
